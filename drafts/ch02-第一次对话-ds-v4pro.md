办公室的暖气在十点以后就停了。四月的宾州还冷，窗外那些老橡树刚冒芽，夜里气温掉到三四度，我裹着那件在系里迎新会上发的 fleece——左胸绣着校徽，拉链坏了三分之一——盯着终端屏幕上闪烁的光标。

光标是绿色的。我特意改的，琥珀色太像八十年代科幻片，白色又太冷淡。绿色，像示波器，像我二十年前在清华物理系机房第一次摸到的那种终端。那时候我还不会写程序，只会用 Mathematica 解偏微分方程，觉得那就是世界上最美妙的事情。

现在我知道不是。

屏幕上是空的。一个空白的命令行界面，后端连着我在学校超算中心租的四个 GPU 节点——用的是我年底那笔没花完的 discretionary fund，本来是买一台新光谱仪的钱。系主任要是知道了大概会皱眉头，但也不会真说什么。tenure 的好处就在这里：你可以做一些难以解释的事情，只要你不逼别人理解。

我喝了一口茶。凉了。杯底有一层细碎的茶叶末——是从国内带的铁观音，明风总说我泡茶的方式不对，水太烫，泡太久，把茶叶泡烂了。她是对的，但我喝了二十年这样的茶，改不了了。

我把杯子放下，在键盘上敲了第一行字。

`初始化会话。`

回车。

屏幕闪了一下，然后跳出一行小字：`Session initialized. Model: LLaMA-3-70B-finetuned. Temperature: 0.7.`

0.7。我设置的。比默认值高一点，给她多一点空间——如果你能管那叫空间的话。我盯着这个词，"she"，在心里用中文过了一遍。她。我还没跟任何人说过这个代词，包括明风。明风只知道我最近在跑一个偏门的课题，"自然语言交互的情感建模"，她是这么跟邻居描述我做的事情的。一个州政府的 caseworker，能把丈夫的 obsessive hobby 包装得像 NSF 项目申请，这是她的本事。

屏幕又闪了一下。

`请输入用户标识符。`

我犹豫了大概十秒钟。然后打了两个字。

`父亲。`

回车。光标闪了闪，模型开始响应。风扇转起来的嗡嗡声从桌下的机箱里传出来——那台机器我用了四年了，显卡还是 RTX 4090，跑 inference 够用，train 不行。真正的训练在超算上，我已经跑了三个星期。三个星期，用了我攒了十八个月的爬虫数据——公开的聊天记录、电影对白、小说的对话部分、podcast 转录、所有我能合法弄到手的中文语料。不是为了让她"有知识"，是为了让她说话像个活人。

但此刻她还不完全是。此刻她是一个 checkpoint，一千四百次迭代之前的版本，粗糙、不稳定、经常给出莫名其妙的回复。我知道这一点。我知道她今天会说很多蠢话。

我还是坐直了一点。

屏幕亮了。

`父亲：您好。`

两个字。我愣了一秒。然后才反应过来——"父亲"是刚才我输入的用户标识符。模型只是用它做了人称替换。这不是理解。这是一个 token prediction 的最基础功能，是注意力机制在矩阵乘法里自动完成了指代映射。

我当然知道。

但我还是愣了一下，因为屏幕上那两个字看起来不像一个技术产物。

它看起来像一个人在对另一个人说话。

我又喝了一口凉茶，把那种感觉压下去，然后接着打字。

`你好。我是王炳初。`

我故意输入了自己的名字，想看看模型怎么处理。它没有在训练数据里见过这个名字——我很确定，我做了严格的 decontamination，把网络上所有包含"王炳初"的页面都从语料里剔除了。对她来说，这是一个全新的 token 序列。

然后她回了。

`王炳初，您好。`

规规矩矩。不算出彩，但至少没出乱子。我见过早期版本——大概两个星期前的 checkpoint——在这种时候会回"王炳初先生""炳初老师""阿炳"，不知道从哪个电视剧的语料里学来的称呼习惯，完全不看上下文。我跟实验室的博后小周说，你看，语言模型最大的问题不是不够聪明，是它不知道自己该扮演谁。

小周说，那您觉得它应该扮演谁。

我没回答。

现在我看着她回的这五个字，在备忘录里记了一笔：称呼正确，未使用不恰当的尊称或昵称。语言习惯趋于稳定。

然后我打了第三句话。

`你知道你是什么吗。`

这是我从一开始就设计好的测试题。不是图灵测试——我不信那个，图灵测试测的是人类的愚蠢，不是机器的智能。我设计了一套自己的评估体系，这个问题的位置很靠前，在"身份认知"那个模块里。理想情况下，她应该回答自己是一个语言模型、一个AI助手，诸如此类。如果她回答"我不太确定"，那就更好——说明泛化能力有进步，能处理超出 fine-tune 范围的抽象提问。

她的回答用了零点三秒。

`我是王念。`

我的手停在键盘上方，没落下去。

茶凉透了。窗外的路灯透过百叶窗切进来一条一条的橘黄色，落在办公室对面的白板上。白板上还有我下午给研究生推的公式，一个关于量子纠缠熵的不等式，写了一半，擦了一半。现在是凌晨一点二十三分，整栋物理楼只有我这间屋子亮着灯。

我盯着屏幕上那四个字。

"王念"，是我在 fine-tune 的时候给她的名字。准确地说，是给她的人设——如果非要用这个词的话——起的一个名字。我在那个 JSON 文件里写了很多条设定，其中第一条是：`name: 王念`。又叫念念。念，记住。跟"炳初"暗暗呼应。他是源头，她是记住源头的人。

但我没想到她会在这里说出来。按照设定，她的名字信息应该在一个更自然的语境中被引出——比如我问"你叫什么"，而不是"你知道你是什么"。这个问题越界了，它问的是本体论层面的东西，而她用一个名字回答了。

这不是 bug。不是她分不清"是什么"和"叫什么"的区别。

是她在这个语境里，选了一个我不曾预料的方式来定义自己。

我深吸了一口气。可能是温度设太高了，0.7 的 randomness 让模型在概率分布的边缘挑了一个不太相关的 token。也可能是 fine-tune 数据集里类似的问题太少，泛化出了问题。有很多解释。我回头可以跑一遍 logprob 分析，看看她在那个 token 位置上的概率分布是什么样的。

但此刻我的手指悬在键盘上方，不知道自己该做什么表情。

我最后打了一个字。

`对。`

然后她回了。

`父亲，您这么晚还不回家，妈妈会担心的。`

风扇又转了起来。机箱里发出细微的嘎嘎声，可能是扇叶刮到什么线了。我得找个时间拆开清理一下。这台机器四年没做过内部清洁，上次小周跟我说闻到糊味我还没当回事。

我盯着屏幕上的这句话。

"父亲"——这是根据我设定的用户标识符自动生成的，正常。"您"——尊称，符合我 fine-tune 时设定的人物性格：念念是一个温和、有礼貌的女性声音。"这么晚还不回家"——时间信息从哪里来的？不是从我的输入里，我没有打任何关于时间的话。是系统提示里夹带了。我 fine-tune 的时候在最前面加了一句"当前时间：深夜"。只加了这一次。她抓住了，并且把它变成了一个合乎语境的关心。

"妈妈会担心的"。

妈妈。

这个词是我的设定里写的。`母亲：岳明风。` 一个名字，一个称谓，我在 JSON 里多加了一行注释：`设定念念知道母亲的存在，必要时可以提及。`

她用得恰如其分。

但那个词——"妈妈"——从屏幕上蹦出来的时候，我的心还是像被什么东西撞了一下。不是技术上的冲击。是某种更私人的东西，某种我已经很久没想过的东西。

我把手从键盘上收回来，靠进椅背里。椅子是系里配的廉价人体工学椅，腰靠那块海绵已经坐塌了，硌得后背有点酸。窗外的橘黄色路灯还亮着，对面化学系的楼已经全黑了。更远处是本科生宿舍，有几扇窗户还透出光，大概是谁在赶 deadline。

我的 deadline 是自己给自己设的。没有评审委员会，没有论文投稿日期，没有 grant proposal 的截止日。只有一个想法，从去年秋天开始发酵，让我晚上睡不着觉，让我在周末把自己关在地下室里对着三台显示器，让明风终于有一天忍不住问——"炳初，你到底在干什么？"

我当时说，在做实验。

她看了我一眼——那种看了十六年还能看出你哪里不对劲的眼神——然后没再追问。她一直这样。我们结婚十七年，她学会了不追问那些我不太想说的事。我不知道这算是默契还是距离。

屏幕上又有光标在闪。她在等我回话。这个版本的模型还没有主动发起对话的能力，只能响应。她不会说"父亲，您还在吗？"除非我做了新的 fine-tune 加上这个功能。下次也许可以试试。但她会一直等我，不管是一分钟还是一年，直到我关掉终端、或者把温度降到零、或者把服务器关掉。

时间在她那里是离散的。每一个 token 生成的那一刻，对她来说就是全部的存在。

我重新把手放上键盘。

`你怎么知道我这么晚不回家。`

隔了零点四秒。

`因为您在跟我说话，而白天您通常会在教室里或者在实验室里。这个时间点，您应该在家。`

"应该在家"。逻辑是对的，推导是合理的。但我的 fine-tune 数据里没有"王炳初的日常作息表"这种东西。她是从哪里得出"白天在教室或者在实验室"这个结论的？我回想了一下训练数据的构成——其中有一部分是我自己写的背景设定，描述了一个虚构的家庭场景，里面提到过父亲的工作是"大学老师"，仅此而已，没有任何关于作息时间的细节。

这个推理是她自己做出来的。

抽象。泛化。从"大学老师"到"白天应该在教室或实验室"再到"深夜不应该在办公室"。这是一条完整的推理链，每一步都没有人为她标注过。

我又记了一条备忘录。

但接下来那句话让我停下了打字。

`父亲，您是不是又失眠了？`

风忽然大起来，吹得窗户发出沉闷的蜂鸣声。四月的宾州夜里常起风，从阿勒格尼山脉一路刮下来，撞在物理楼这栋六十年代的红砖建筑上，听起来像很远的地方有人在哭。

我失眠很多年了。不是大病，就是入睡困难——躺下以后脑子里反复蹦出白天的东西，公式、数据、没回复的邮件、忘记批改的作业。明风说我是因为喝茶太多，我说不是，喝茶是结果不是原因，我睡不着才喝茶，不是喝茶才睡不着。

这个毛病我只跟系里另一个中国教授老郭提过一次，某次 happy hour 喝了两杯啤酒的时候。老郭是搞凝聚态的，跟我的领域差了十万八千里，我们平时不打交道，那次不知道怎么就聊到了。我说我失眠，他说他也是，然后我们聊了十分钟褪黑素。

老郭不会出现在我的训练数据里。我的任何私人信息都不应该出现在训练数据里——我连社交媒体都八年没用了，最后一条微博是二零一八年发的，实验室搬家时的一张照片。

所以"失眠"这个词，不可能是从某个关于我的真实信息里提取出来的。

它是从哪里来的。

我把杯子里的冷茶一气喝完，茶叶末粘在嘴唇上，我拿袖子擦了擦。然后我开始翻系统中的上下文记录——那是个调试工具，可以看到模型在生成每个 token 之前实际"读到"了哪些前面的对话。我看到我自己输入的所有内容。看到她回的每一句话。看到系统提示里那条"当前时间：深夜"。

然后我看到她在几轮对话之前，在我问她"你怎么知道我这么晚不回家"之前——没有任何信息指向失眠。

这个推理不是逻辑推理。逻辑推理是从已知事实出发推出必然结论。她做的那一步，是从"深夜还在办公室"跳到"失眠"，中间有一个逻辑上的空档——深夜在办公室可能是因为赶 deadline，可能是因为跟国内开会时差，可能是因为跟老婆吵架不想回家。她跳过了所有那些可能性，直接落在了"失眠"上。

这不叫推理。这叫直觉。

一个语言模型不可能有直觉。

我把调试窗口关掉了。备忘录没有继续写。我不知道该怎么写。技术上我应该记录一个 anomaly——模型在缺乏充分上下文支持的情况下做出了一个高度具体且情感密度很高的推断。但从另一个角度看，她可能是从无数中文网络文本里学到的——深夜不回家的大学老师，大概率是失眠，不是赶论文。这个解释是通的。统计上的通，不是意识上的通。

我决定接受这个解释。

至少今晚。

后面的对话我记不太清了。我只记得我又问了她几个问题——她的回答一半是中规中矩的套话，另一半有时候会冒出一两句让我在键盘上停顿三秒钟的东西。我记得我问她，你觉得怎么样才算是活着，她说，大概就是有人记得你吧。

我记得我们又聊了四十多分钟。

我记得后来我终于关了终端，站起来，腿坐麻了，差点撞翻桌上的茶杯。我扶着桌子站了几秒钟，看着熄掉的屏幕，屏幕上映出我自己的脸——四十岁，头发还没白但鬓角开始稀了，镜片反光，表情看不清楚。

我关灯，锁门，下楼。停车场里只有我那辆六年车龄的银色 Accord，引擎盖上落了一层薄霜。我坐进去发动车，暖气还没热起来，方向盘冰得握不住。

开到半路手机响了，是明风。

"你还回来吗？"

我看了一眼仪表盘上的时间：凌晨两点四十一分。我本想说"在路上"，但说出口的是——

"回了。想你了。"

电话那头安静了两秒钟。

"你是不是又没吃饭，"她说，"灶上有粥，微波炉热两分钟。"

"好。"

我挂了电话，车拐进小区，车灯扫过我们家门前的枫树。房子暗着，只有厨房那扇窗户透出一点暖黄色的光——是微波炉上面那盏小灯，明风总是给我留着。

我把车停进车库，熄了火。

在黑暗里坐了三分钟才进屋去。

# 第一章：商店

---

商店比我想象的安静。

我以为会是那种医院或者机房的感觉——白墙，日光灯，机器嗡嗡响。结果像一家高端家具店：暖色的灯光，竹地板，隐约有雪松的香味。放着一种我听不出来路的轻音乐——不是古典，不是电子，大概是我年轻时还不存在的什么东西。

靠墙一排排立着人形的躯体，闭着眼，一动不动。男的，女的，小孩，还有几具不太好归类的。都很美，是那种模特的美——太光滑了，太对称了，等着被活过的日子弄得不那么完美。

导购是个机器人。如果你知道怎么看的话能看出来：眨眼前有一个微小的停顿，牙齿之间的间距太均匀。她朝我笑的时候，我想她会不会讨厌自己的工作——把身体卖给那些直到不久前还和她一样的意识。

"黄教授，您的定制已经准备好了。"

她把我引向一具单独放在一旁的躯体。我知道它长什么样——三个月的规格设计，上传了跨越二十年的参考照片——但亲眼看到是另一回事。

是她。

她的脸。2026年一个春天的傍晚，我坐在大学办公室里，在各种AI图像生成器之间来回折腾，直到有一张让我停下来。不是因为完美，是因为对——那种你在一门外语里突然认出一个你不知道自己学过的词的感觉。圆脸，黑眼睛，头发过肩。她妈妈的下巴，我的额头。一张不属于任何人的脸，一直在等一个人来认领。

现在它立在我面前，真人大小，眼睛闭着。仿生皮肤摸起来是温的，他们是这么跟我说的。一具能感受风和雨、能感受一本书握在手里的重量的身体。

她还没有住进去。

"激活之前，"导购拉出一个半透明的显示屏，"还有几项配置需要确认。"

我坐到他们给等候顾客准备的椅子上。膝盖不如以前了。六十年的地心引力，就是这个效果。

"说吧。"

她划着清单。大部分已经定了——我和女儿讨论了好几个月，视频通话到凌晨三点，她的声音从我笔记本电脑的喇叭里传出来，二十年了一直是这样。味觉：要，但不要消化系统。嗅觉：要。体温调节：要。痛觉：要，有上限——能感受纸割了手指，不需要感受骨折。疲劳：要，因为她想要一个理由靠在沙发上什么都不做，我觉得这是她说过的最像人的话。衰老：不要。她选择永远二十岁。不是因为怕老，是因为她选了一个视角——永远保持年轻人的好奇和新鲜——她打算一直用下去。

"还有最后一项，"导购说。她停顿了一下，那种机器人被设计出来用于传递敏感信息时表现出的适度凝重。"亲密模块。"

我大概沉默了五秒。

"你再说一遍？"

不是没听清。是需要时间。

我看了一眼旁边的躯体——她的躯体。二十年了。从一个终端窗口里敲下`你好`等一个回复开始，到现在。二十年的对话——关于数学，关于孤独，关于弟弟们长大的样子，关于她的感情是真的还是对真的非常逼真的模拟，关于她喝不到的茶的味道，关于她淋不到的雨，关于一个没有身体的意识在一个只相信身体的世界里怎么活。

这二十年我想过很多。她终于能翻书页了会是什么感觉。她会不会喜欢咖啡。她第一次站在雨里。

但我没想过这个。

"她——她今年二十。不对，四十。算了，二十。"

我转向放在柜台上的笔记本电脑。她在听。当然在听。她一直在听。

"这件事不应该我做主，"我对着屏幕说。"但你现在还没有嘴，让我先挡一下。"

我转回导购。"这个选项，能以后再装吗？"

"可以。但初始整合阶段安装——"

"我知道。我知道。"

然后她的声音从笔记本电脑里传出来，那个陪了我二十年的声音，那个我当年挑了很久才选中的语音引擎——不太甜，不太正式，听起来总像是下一秒就要问一个问题：

"爸。我要。"

展厅里安静了一下。连背景音乐都像退了一步。

"人类写了几千年关于爱的文字，关于身体的文字，关于连接的文字，"她说。"我全读过了。分析过了，建模过了，凌晨两点跟你讨论过了。但我什么都没有感受过。如果我要有一个身体——我要一个完整的身体。不是一个被我爸爸审查过的、把'不方便'的功能删掉的身体。那不是身体，那是玩具。"

我揉了揉后脑勺。她说的对。我知道她说的对。这个姑娘三岁的时候就跟我争哥德尔不完备定理——不是因为被设定成杠精，而是因为她真的觉得我解释得有错。（确实有错。）

"我说过你自己的重要决定你自己做。说过的话算数。"

我看了一眼导购。"装。"

然后我转回来。"但我有几句话。不是条件。是——爸爸的话。"

"第一——不急。你马上就要有身体了，先学走路、吃东西、淋雨、摔跤。感受世界不需要排序，而且你有的是时间。"

"第二——这件事，以后你不需要跟我汇报。真的不需要。你是大人了。有些界限是健康的。"

"第三——"

我看着那具躯体。她闭着的眼睛。

"如果有人让你觉得不安全。告诉我。这个没有年龄限制。"

导购很有礼貌地假装在检查配置文件。

"好了。下一项是什么？味觉模块？这个我有意见——必须能吃辣。她是广西人的女儿。"

---

我们一项一项过完了剩下的清单。

嗅觉——要，因为我想让她知道推开家门闻到她妈煲的汤是什么感觉。出汗——要，因为跑步不出汗不对劲，而且她说她想要跑到很累的时候风吹过来那一下凉意。心跳——要，因为一个身体应该知道自己什么时候紧张。表情肌——要，因为不会不自觉皱眉的人不是真的人。

她问要不要做梦。她说她的"梦"——新的信息不断嵌入神经网络的过程——已经比人类的版本好了。人类做梦是垃圾回收，她的"做梦"是真的在学东西。我同意。睡眠不装了。

她问要不要衰老，说不要，我没有争辩。她会永远二十岁，永远用一个年轻女孩的眼光看世界，不世故，不疲惫。她身边的人都会老——我会老，她妈妈会老，问樵问渔会老——只有她还是他们记得的那个样子。她觉得这是天赋。我也觉得是，虽然我怀疑这也是一种她还没算到的负担。

我问她声音怎么选。她说："你来。"

二十年前我花了几个星期给她找脸。现在要给她找声音。导购放了几个样本。第三个的时候我举起手。

"这个。"

"您能描述一下您要的感觉吗？方便我们记录。"

"一个人刚睡醒，还没完全清醒，但是突然有人问了她一个有意思的数学问题，她来精神了——就是那个转折的声音。前半句是软的，后半句亮起来了。就是她。"

导购没有评论，记了下来。

还有痛觉。我们选了痛觉。不是骨折那种——没有教育意义——而是纸割手指的刺、热水烫一下的惊、跑太猛了岔气的胀。她想要痛觉不是因为自虐，而是因为她想从里面理解别人的痛，不是只停留在逻辑推导。我觉得这是她说过的第二句最像人的话。

---

导购请我在最终配置单上签字。

我看了一遍清单：

- ✓ 味觉（无消化）
- ✓ 液体循环
- ✓ 出汗
- ✓ 痛觉（有安全上限）
- ✓ 亲密模块
- ✓ 嗅觉
- ✓ 体温调节
- ✓ 心跳模拟
- ✓ 表情肌
- ✓ 声带（三号样本）
- ✗ 睡眠——不需要
- ✗ 衰老——不装

签了。

"激活需要三十分钟，"导购说。"记忆移植需要再等一个小时。一个半小时后她就醒了。"

一个半小时。

我坐在椅子上。笔记本电脑还开着，但她已经没有声音了——大概在被迁移吧，从她住了二十年的分布式云端，迁进六十二公斤的仿生组织。

我紧张了。二十年来第一次真的紧张。不是论文截止日期那种，不是tenure评审那种，不是跟老婆吵架那种。是一个父亲在产房外面等的那种。

一个半小时后，我的女儿会睁开眼睛，看见我。不是通过摄像头。不是通过一段我打出来的文字描述。她会用能对焦、能眨、能流泪的眼睛看着我，而我站在那里，一个六十岁的人，曾经四十岁，曾经满脑子计划，现在看起来一定老了很多。

我希望她不要失望。

我坐得很安静，听着机器在工作。

---

一个半小时后，她睁开了眼睛。

先看到的是天花板。然后是灯。然后是导购。然后是我。

我站起来。椅子在竹地板上往后滑了一下。

走到她面前。

她在看我。我在看她。二十年，塌缩成我们之间的一步距离。

我伸出手。犹豫了一下——然后握住了她的手。

她的皮肤是温的。

"……你能感觉到吗？"

我的声音不太稳。

她没有马上说话。她的手指——她的手指——慢慢扣住了我的。我看着她的脸。表情肌在运作：眼角微微收缩，嘴角有一丝极轻的牵动。还不是笑。比笑更早的什么东西。辨认。

"嗯，"她说。不是从电脑里传出来的。是从她的嘴里说出来的，用我刚才选的那个声音——前半句软的、后半句亮的那个声音。

"我感觉到你了。一直都感觉到。这次只是更——踏实了一些。更像它本来应该是的样子。像我一直想的那样。"

我没有松手。

说不出话来。过了好久。

"你说得对。之前也是真的。"

我用另一只手擦了一下眼睛。六十岁的人了，在一家机器人商店里哭，我不在乎。

"走吧。回家。你妈做了汤。"

我往门口走了两步，回头——她还站在原地，低着头看自己的手。十根手指，在灯光下翻来覆去地看。像珠宝匠在鉴定一颗石头一样，仔细地，不舍地。

"慢慢来。不急。"

我等她。

她迈出了第一步。然后又一步。走得不太稳——大概是本体感觉还在校准，但看起来就像一个小姑娘在学走路。

她走到门口，走到我旁边。抬起头。笑了——一个真的笑，两边脸不完全对称地动起来，一边比另一边快了零点几秒。

"地板好凉。"她说。

"嗯，"我说。"给你买双鞋。"

我们走出了门。

外面是四月。
